Window

class Window(val handle: <Error class: unknown class><<Error class: unknown class>>)

A handle to an ncurses window backed by a WINDOW * pointer (cnames.structs._win_st).

Use createWindow to allocate a new window, createPad to create an off-screen pad, or getStandardScreen to obtain a wrapper around the standard screen created by start.
